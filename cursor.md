# Whiskritório — Design System & Project Rules

## Stack

* Next.js App Router
* TypeScript
* Tailwind CSS
* shadcn/ui
* Framer Motion

## Visual Identity

Whiskritório is a convenience store and distributor (Conveniência e Distribuidora).

Style:

* cinematic
* premium
* modern
* dark
* neon
* clean retail aesthetic

Main colors:

* zinc-950 background
* black
* neon yellow accents
* subtle white typography

Avoid:

* generic layouts
* excessive gradients
* colorful palettes
* outdated UI

## UI Style

Inspired by:

* iFood premium
* Uber Eats
* modern convenience store apps
* premium retail e-commerce

Use:

* glassmorphism lightly
* soft shadows
* neon glow effects
* smooth hover transitions
* premium spacing
* rounded-2xl
* strong typography hierarchy

## Animations

Use Framer Motion for:

* fade-ins
* stagger animations
* hover effects
* scroll reveal
* smooth transitions

Avoid excessive or flashy animations.

## Components

Create reusable components:

* HeroSection
* Navbar
* ProductCard
* EventSection
* GallerySection
* Testimonials
* CTASection
* Footer

## Responsiveness

* mobile first
* perfect mobile spacing
* responsive typography
* optimized images
* smooth experience on low-end devices

## Code Rules

* clean code
* reusable sections
* semantic HTML
* accessibility friendly
* optimized performance
* avoid unnecessary dependencies

## Goal

The website must feel:

* luxurious
* modern
* immersive
* visually impressive
* highly premium
